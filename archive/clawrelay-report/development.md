# Clawrelay Report Development

## 对人使用的唯一入口

- 管理后台：<http://localhost:5173>

## 内部服务

- 后端 API：<http://localhost:8000>

注意：

- `8000` 仅作为前端依赖的 API 服务
- 不把 `8000/docs`、数据库管理页、代理面板等写成并列后台入口

## 本地开发约定

1. 前端入口固定为 `http://localhost:5173`
2. 后端服务固定为 `http://localhost:8000`
3. 所有对外说明都应把 `5173` 作为唯一后台入口

## 范围约束

本项目是项目标准中的唯一管理后台。

不在本项目标准中的内容：

- 把 Grafana 当作主后台
- 把 SSE 当作已确认能力
- 引入与当前架构不一致的平行执行路径
